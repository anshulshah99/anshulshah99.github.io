Projects

Teaching Incremental Software Development through "Live Coding"

Gerald Soosairaj, Bill Griswold, Leo Porter

My project focuses on promoting effective software engineering practices such as frequent testing and incremental coding by utilizing "live coding" during lectures. Live coding is a teaching practice where an instructor shows the entire programming process live in front of a class, from opening a new file, writing a bit of code, testing the code, debugging, and then continuing in this incremental process.

July 2021 - present

Improving Student Metacognition through CompassX

Soohyun Liao

I'm designing a website for the a data structures and algorithms class that is similar to my previous work on the CS1 Reviewer App. I handle all steps of the website creation process, from backend database work to frontend user design. Our project aims to improve student metacognition- how well a student recognizes which concepts they understand and do not understand. To compute metacognitive accuracy among students, we ask students to input a confidence score for each answer they submit. A key aspect of the project is designing an effective dashboard for students to view their metacognitive accuracy and to improve their study habits to focus on the concepts they struggle with.

December 2011 - March 2013

CS1 Reviewer App

Kristin Stephens-Martinez

I created a website for the introductory Python course at Duke University where students could customize their own study quizzes based on a variety of topics and view a summary of their responses in a personalized dashboard. The website estimated student knowledge using the Bayesian Knowledge Tracing algorithm, and used these calculations to provide autogenerated quizzes to students to target the concepts that are more difficult for them.
